HTML代码规范

Posted by Nutlee on 2016-04-28

 

标签闭合，尽量少用

class 应以功能或内容命名，不以表现形式命名； 命名小写，避免无样式class

HTML标签内属性顺序

  • id
  • class
  • name
  • data-xxx
  • src, for, type, href
  • title, alt
  • aria-xxx, role

属性用双引号

严格嵌套

HTML5 规范中 disabled、checked、selected 等属性不用设置值

HTML HEAD

  • <!DOCTYPE html>

  • 字符编码

  • 优先兼容最新版本的IE和Chrome浏览器

  • 完整HEAD
