关于 iOS，你可能不要知道，但又必须要知道的

Network Extension

能干嘛？

在 iOS9 以前，翻墙是件很困难的事，只能通过 Wi-Fi 时设置代理，安装描述文件，或者忍受。

2015 年的 WWDC，也就是 iOS9 的发布会上，迎来了这个神奇的 API。
需要在 Apple Store Developer 中心申请
被 Potatso 的开发者称为 “iOS 史上最为开放的 API”

而现在我们可以依赖这个 API 接管整个手机网络，甚至可以用来干掉广告。

各类翻墙黑科技
都是基于 Shadowsocks 项目

  • Surge (Mac/iOS)
  • Shadowrocket
  • Potatso(1.0 已开源)

ShowTime(Surge 演示 用手机抓 新浪微博)
